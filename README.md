# MyWordle

## 1. Описание проекта

**MyWordle** — это мультиплеерная веб-реализация игры Wordle с поддержкой русских слов.  
Приложение позволяет нескольким пользователям участвовать в одной игровой сессии, соревнуясь в угадывании слова.

Система построена по клиент-серверной архитектуре:

- **Backend:** FastAPI (Python)  
- **Frontend:** React + Vite  
- **Связь в реальном времени:** WebSocket  

Приложение поддерживает создание игровых лобби, подключение игроков по коду и различные режимы игры.

---

## 2. Архитектура системы

Система состоит из двух основных компонентов:

### Клиентская часть

Клиентская часть реализована как веб-приложение на **React**.  
Она отвечает за:

- отображение игрового интерфейса  
- ввод слов игроком  
- отображение статусов букв  
- отображение прогресса игроков  
- взаимодействие с сервером через REST API и WebSocket  

Для сборки и разработки используется **Vite**.

### Серверная часть

Серверная часть реализована на **FastAPI** и отвечает за:

- управление игровыми лобби  
- обработку подключений игроков  
- выбор случайного слова  
- проверку вводимых слов  
- синхронизацию состояния игры между игроками  

Для обмена данными используется формат **JSON**.  

Игровые данные (слова) хранятся в **базе данных**, что позволяет эффективно выполнять выбор слов по заданной длине и проверку корректности вводимых пользователями значений.

---

## 3. Основные возможности

Приложение MyWordle поддерживает следующие функции:

- создание **игрового лобби по коду**  
- подключение **до 4 игроков**  
- выбор **длины слова**  
- использование словаря **русских слов**, хранящегося в базе данных  
- несколько режимов игры:
  - классический режим  
  - режим **гонки на время**  
- **командный режим** (команды A/B) с общим счётом  
- синхронизация состояния игры через **WebSocket**  
- проверка слов с выдачей статусов букв:
  - **correct** — буква находится на правильной позиции  
  - **present** — буква есть в слове, но на другой позиции  
  - **absent** — буква отсутствует в слове

## 4. Запуск проекта

### Серверная часть (FastAPI)

```
cd backend
python -m venv .venv
```

Активация виртуального окружения:

```bash
Windows:
.venv\Scripts\activate
Linux/macOS:
source .venv/bin/activate
```

Установка зависимостей:

```
pip install -r requirements.txt
```

Запуск сервера:

```
uvicorn app.main:app --reload --app-dir backend --host 0.0.0.0 --port 8000
```

Сервер будет доступен по адресу: http://localhost:8000

Клиентская часть (React + Vite)

```
cd frontend
npm install
npm run dev
```
